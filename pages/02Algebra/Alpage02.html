Propiedades en la Intersección:

La intersección de Conjuntos cumple las propiedades;

Clausurativa: An A = A

Asociativa: An (Bn C) = (An B)n C

Intersección con el Conjunto Vacio: An Ø = Ø

Absorción: Si ACB => AnB = A

Conmutativa: An B = BNA

Propiedades en la Unión:

La Unión de Conjuntos cumple las propiedades;

Clausurativa: AU A = A

Asociativa: AU (BU C) = (AU B)U C

Unión con el Conjunto Vacio: AU Ø = A

Absorción: Si ACB => AUB = B

Conmutativa: AU B = BUA

Propiedades Combinadas:

La intersección de Conjuntos cumple las pro piedades;

Distributiva respecto de la Intersección: AU (BN C) = (AU B)n (AUC)

Distributiva respecto de la Unión: An (BU C) = (AN B)U (ANC)

Simplificativa respecto de la Intersección: AU (BN A) = A

Simplificativa respecto de la Unión: An (BU A) = A

Relaciones entre Conjuntos:

•REFLEXIVA: SI VXE A, (x, x) E AX A

•SIMÉTRICA: SI V (x,y) € AxA => (y,x)E AxA

•TRANSITIVA: Si V(x,y)E AXA,A (y,z)E AxA => (x,z) € AXA

•DE EQUIVALENCIA: Cuando es simultáneamente Reflexiva, Simétrica y Transitiva.

•DE ORDEN: Cuando es Reflexiva, Antisimétrica y Transitiva

Función: Es la relación entre los elementos de un conjunto o la imagen y los de otro conjunto o el dominio. A cada elemento del dominio, corresponde un elemento del primer conjunto sobre el cual es aplicado o representado por la función. - f (A cada elemento x corresponde un único y)

f = [(x,y) / y = f (x)] y: es una función de x

Función Inyectiva: A elementos distintos del dominio tiene una única imagen en el rango; ningún elemento del conjunto B posee más de un original.

Función Sobreyectiva: Cada elemento del rango es imagen de uno o más elementos del dominio o conjunto Original.

Función Biyectiva: Cuando es inyectiva y sobreyectiva al mismo tiempo; cada imagen tiene un único original y cada original tiene una sola imagen.
