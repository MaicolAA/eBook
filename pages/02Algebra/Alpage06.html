Teoría de Conjuntos

Cada uno de los objetos que forman parte de un conjunto se llama elemento. Un conjunto se define por extensión cuando se enumeran todos y cada uno de los elementos que lo forman, y se define por comprensión, cuando se enuncian todas las propiedades comunes de sus elementos. Para representar los conjuntos se usan letras mayúsculas en tanto que su elementos se suelen denotar con letras minúsculas. Para representar gráficamente los conjuntos de usan Diagramas de VENN, que son lineas curvas cerradas con los elementos dentro. Cuando un conjunto no tiene elementos, se dice que es vacío, y se denota: Ø. Cuando los elementos de un conjunto A pertenecen también a otro conjunto B se dice que A es un subconjunto de B, y se indica: ACB Si N es el conjunto de los números Naturales y P es la propiedad "ser múltiplo de 5", los números que cumplen esta propiedad se denotan: {x| x es múltiplo de 5}. Dados dos conjuntos A y B se llama conjunto complementario de A con respecto a B al conjunto formado por todos los elementos de B que no pertenecen a A. Al conjunto que contiene todos los elementos de dos o más conjuntos se le llama unión.
