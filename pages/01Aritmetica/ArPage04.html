Propiedades: la multiplicacion de numeros naturales cumple las propiedades: UNIFORME; pues el producto de dos numeros naturales es siempre un numero Natural.

La PROPIEDAD ASOCIATIVAmuestra que: a x (b x c) = (a x b) x c En la PROPIEDAD CONMUTATIVA se cumple que a x b = b x a

El ELEMENTO NEUTRO del Producto de Números Naturales es el número '1' al cumplir que: a x 1 = 1 x a = a La PROPIEDAD DISTRIBUTIVA para los Números Naturales cumple: a x (b + c) = (a x b) + (a x c).


La División de Números Naturales es la operación inversa a la Multiplicación y busca hallar uno de los factores llamado COCIENTE, conociendo otro factor llamado DIVISOR y el producto, que se conoce como DIVIDENDO. Entre el Dividendo y el Divisor se coloca el signo (÷) ó (/). Se le llama RESIDUO a la cantidad que sobra en la División.


Los Números
Enteros 'Z'

Un Número entero de compone de un SIGNO que puede ser POSITIVO o NEGATIVO y de un Número Natural. El cero '0' también se incluye en los Números Enteros:

Z = [..., -3, -2, -1, 0, 1, 2, 3... ]

Suma y Propiedades: En los Números Enteros: Se cumple la PROPIEDAD ASOCIATIVA: a + (b +c) = (a +b) + c

La PROPIEDAD CONMUTATIVA verifica que:

a + b = b + a El ELEMENTO NEUTRO en la Suma de Números Enteros es el '0' al

Verificarse que: a + 0 = 0 + a = a

Para cualquier Número Entero 'a' existe un Número Entero 's' o SIMÉTRICO tal que:

a + s = s + a = 0, donde s = -a

Resta de Enteros: Para restar dos Números Enteros basta con sumar al primero el opuesto del segundo; es decir, si tomamos los pares (a,b) y (c,d) para efectuar la resta tendremos entonces:

( a , b ) - ( c , d ) = ( a , b ) + ( d , c ) = ( a + d , b + c)
