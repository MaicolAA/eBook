Los Números
Naturales

Es una sucessión con infinitos términos:
N: 0, 1, 2, 3, 4, 5, 6, 7, 8 y 9.
La Suma de Números Naturales es la operación que une en un solo número las unidades de varios números. Los numeros sumados son los SUMANDOS y la operación se representa con el sigo MÁS (+).
Propiedades: La Suma de los Números Naturales cumple la propiedad: UNIFORME; pues la suma de dos Números Naturales es siempre un Número Natural. Cumple la propiedad ASOCIATIVA: a + ( b+C ) = (a+b) + c Tambien es CONMUTATIVA: a + b = + a
El "0" se considera el ELEMENTO NEUTRO o MÓDULO en la Suma de Números Naturales al cumplir que: a + 0 = 0 + a = a
La Resta De Números Naturales es la operación opuesta a la Suma; dinde a un número llamado MINUENDO se le quita otro llamado SUSTRAENDO.
